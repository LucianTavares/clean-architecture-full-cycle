# Módulo de Clean Architecture - Full Cycle.

## Execução do projeto:

1 - Clone o repositório:
- `git clone https://github.com/LucianTavares/clean-architecture-full-cycle.git`

2 - Mude para a pasta:
- `cd clean-architecture`

3 - Baixe as dependências e execute os testes: 
- `npm install`
- `npm test`
